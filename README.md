## 1. Project Identity & Philosophy

### 🔭 Vision: "Zero-Hallucination AI Archive"
세상의 모든 지식을 담겠다는, 누가 봐도 불가능해 보이는 아카이빙 시스템. 단순히 글을 보여주는 블로그를 넘어, 내가 직접 검증하고 기록한 Notion 데이터만을 소스로 삼아 향후 AI 에이전트(RAG)가 100% 신뢰할 수 있는 답변을 도출해 내는 '무결성 지식의 요람'을 지향합니다.

### 💎 Core Values
- **Verification (검증):** 직접 경험하고 느낀 것들을 나만의 체계화 과정을 거쳐 하나의 책처럼 발행합니다.
- **Automation (자동화):** Notion에 글을 쓰는 즉시 웹사이트에 동기화되며, 궁극적으로는 지식을 통한 가치 창출(수익화) 시스템과 연결됩니다.
- **Aesthetics (미학):** 고풍스러운 서재의 무드와 현대적인 Glassmorphism(유리 질감) UI를 융합하여 독보적인 분위기를 연출합니다.

## 2. Key Features

### 💀 Archive Librarian (지능형 사서 챗봇)
- **Interactive Persona:** 서재를 지키는 마스코트 '해골 사서'가 챗봇 UI로 상주하며 방문자를 맞이합니다.
- **Real-time Interaction:** 모서리가 둥근 글래스모피즘 모달창과 매끄러운 타이핑 애니메이션으로 깊은 몰입감을 선사합니다. (향후 RAG 시스템의 핵심 프론트엔드 역할을 수행)

### 🗂️ Dynamic File Tree & Fast Search
- **Notion as a DB:** 노션의 복잡한 하위 페이지 구조를 재귀적으로 파싱하여 탐색기(File Tree) 형태로 시각화합니다.
- **Instant Search:** 깊은 뎁스(Depth) 속에 숨겨진 문서라도 클라이언트 사이드에서 즉시 찾아냅니다.

### 💬 Interactive Request System
- **Discord Webhook:** 사용자가 새로운 지식을 요청하거나 피드백을 남기면, 관리자의 Discord 채널로 즉시 알림이 전송되는 파이프라인을 구축했습니다.

## 3. Tech Stack & Architecture

### Frontend & UI
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS v4 & Framer Motion
- **State Management:** Custom Hooks (`useChat` 등)를 활용한 비즈니스 로직 분리

### Backend & Infrastructure
- **CMS:** Notion API
- **Deployment:** Vercel

## 4. Ideation Process: Visual First (시각적 프로토타이핑)

Dechive의 모든 새로운 기능과 디자인은 무작정 코드부터 타이핑하는 것이 아닌, 확실한 **시각적 시뮬레이션**을 거쳐 탄생합니다.

1. **상상과 시각화:** 무언가를 더 만들고 꾸미고 싶을 때, 가장 먼저 AI를 활용해 머릿속의 아이디어를 뼈대가 되는 기본 이미지로 생성해 냅니다.
2. **멘탈 시뮬레이션:** 완성된 디자인 시안을 눈으로 확인하고, 머릿속으로 전체적인 UI/UX 흐름과 사용자 상호작용을 먼저 돌려보며 완벽한 그림을 그립니다.
3. **코드 구현:** 시각적 검증이 끝난 확고한 설계도를 바탕으로, 현실의 코드로 구현합니다.
